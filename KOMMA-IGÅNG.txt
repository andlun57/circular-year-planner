┌───────────────────────────────────────────────────────────────────────┐
│ SNABBINSTALLATION (3 STEG)                                           │
└───────────────────────────────────────────────────────────────────────┘

1. Kopiera hela mappen till: /wp-content/plugins/
2. Aktivera pluginen i WordPress admin
3. Gå till "Årsplanering" i menyn och börja skapa händelser!

┌───────────────────────────────────────────────────────────────────────┐
│ VAD KAN PLUGINEN GÖRA?                                               │
└───────────────────────────────────────────────────────────────────────┘

✅ Cirkulär kalender med månader, veckor och händelser
✅ Händelsenamn visas direkt i cirkeln längs bågen (SVG textPath)
✅ Endagshändelser renderas som hela veckan (bättre synlighet)
✅ Ramar runt varje händelsetypsring för tydlig separation
✅ Automatisk textfärg (svart/vit) för bästa kontrast
✅ Anpassningsbara händelsetyper med egna färger
✅ Flexibelt verksamhetsår (kan börja vilken månad som helst)
✅ Svenska veckonummer (ISO 8601)
✅ Klickbara händelser med detaljvy
✅ Navigation mellan år
✅ REST API för integrationer
✅ Responsiv design (fungerar på mobil, tablet, desktop)

│ FUNKTIONER I DETALJ                                                   │
└───────────────────────────────────────────────────────────────────────┘

🎨 ADMIN-FUNKTIONER:
   • Skapa och redigera händelser
   • Anpassa händelsetyper och färger
   • Välj verksamhetsårets startmånad
   • Välj färgschema för kalendern

📅 KALENDER-FUNKTIONER:
   • Yttersta ringen: Månader
   • Andra ringen: Veckonummer
   • Inre ringar: Händelser (en ring per typ)
   • Centrum: Verksamhetsår

🖱️ INTERAKTIVITET:
   • Klicka på händelse för detaljer
   • Navigera mellan år med pilar
   • Hover-effekter på händelser
   • Legend visar alla händelsetyper

🔌 INTEGRERING:
   • REST API: /wp-json/cyp/v1/events
   • REST API: /wp-json/cyp/v1/settings
   • Shortcode med parametrar
   • WordPress hooks och filters

┌───────────────────────────────────────────────────────────────────────┐
│ EXEMPEL: SKAPA FÖRSTA HÄNDELSEN                                      │
└───────────────────────────────────────────────────────────────────────┘

1. Logga in på WordPress admin
2. Klicka "Årsplanering → Lägg till händelse"
3. Fyll i:
   - Titel: "Årsmöte 2024"
   - Startdatum: 2024-03-15
   - Slutdatum: 2024-03-15
   - Typ: Välj "Program" (eller annan typ)
   - Verksamhetsår: Beräknas automatiskt! 🎉
4. Klicka "Publicera"
